ATAC-seq 與 CUT&RUN

染色質可及性與蛋白-DNA 交互作用：Tn5、MACS、TF motif、scATAC 一次學完。

Chromatin accessibility and protein–DNA interactions: Tn5, MACS, TF motifs, and scATAC — in one chapter.

技術原理

一、三種技術

🪟 ATAC-seq

高活性 Tn5 轉位酶同時切割與標記 open chromatin。Buenrostro 2013。所需細胞少（50k–500k），3 h 完成 library。輸出：可及性 peaks，反映 enhancer / promoter / TF footprint。

Hyperactive Tn5 transposase simultaneously cuts and tags open chromatin (Buenrostro 2013). Few cells needed (50k–500k), library in 3 h. Output: accessibility peaks reflecting enhancers/promoters/TF footprints.

✂️ CUT&RUN

抗體 + pA-MNase 在原位切下 TF / histone 周圍的 DNA。Henikoff lab 2017。比 ChIP-seq 少 10× 細胞、雜訊更低，已是 TF binding 與 histone mark 的主流。

Antibody + pA-MNase digests DNA around TF/histone in situ (Henikoff lab 2017). 10× fewer cells than ChIP-seq, lower background — now the standard for TF binding and histone marks.

🏷️ CUT&Tag

把 MNase 換成 pA-Tn5：抗體靶向後 Tn5 直接 tagment，連 library prep 一起跳過。更少細胞、更快、可做 scCUT&Tag。

Replace MNase with pA-Tn5: antibody-tethered Tn5 tagments directly, skipping library prep steps. Even fewer cells, faster, enables scCUT&Tag.

⚠️
選型：想看「開了哪些區域」→ ATAC；想看「哪個 TF / histone mark 在哪些位點」→ CUT&RUN（高解析度）或 CUT&Tag（少細胞、scale-up）。傳統 ChIP-seq 只在缺乏抗體效能驗證資料時建議。 Selection: Want "which regions are open?" → ATAC. Want "which TF/histone-mark binds where?" → CUT&RUN (high res) or CUT&Tag (few cells, scale-up). Reserve classic ChIP-seq for cases lacking antibody-validation data.
分析流程

二、Peak calling 與下游

比對與過濾

bwa-mem2 → MarkDuplicates → 移除 MAPQ < 30、blacklist (ENCODE)、粒線體 reads（ATAC 特別重要：Tn5 偏好 mt DNA）。

bwa-mem2 → MarkDuplicates → drop MAPQ < 30, ENCODE blacklist, and mitochondrial reads (critical for ATAC — Tn5 prefers mtDNA).

Tn5 shift

+ strand +4 bp、− strand −5 bp 校正 Tn5 9 bp staggered cut（footprint 分析必做）。

+ strand +4 bp, − strand −5 bp to correct Tn5's 9 bp staggered cut (mandatory for footprinting).

Peak calling

MACS3（取代 MACS2）：ATAC 用 --shift -75 --extsize 150 -f BAMPE --nomodel；CUT&RUN 用 SEACR 或 MACS3 narrow。

MACS3 (replaces MACS2): ATAC uses --shift -75 --extsize 150 -f BAMPE --nomodel; CUT&RUN uses SEACR or MACS3 narrow.

差異可及性

取所有樣本 peak union → 計算 counts in peaks → DESeq2 或 edgeR 跑 condition contrast，產出 DARs。

Peak union across samples → counts in peaks → DESeq2 or edgeR for contrasts → DARs.

Motif / TF / annotation

HOMER findMotifsGenome.pl、MEME-ChIP、chromVAR（scATAC）、ChIPseeker 註釋到 promoter/intergenic/UTR。

HOMER findMotifsGenome.pl, MEME-ChIP, chromVAR (scATAC), ChIPseeker for promoter/intergenic/UTR annotation.

互動 — 片段長度

三、片段長度分佈

高品質 ATAC 應在 ~50, ~200, ~400 bp 看到清楚峰，分別代表 nucleosome-free、mono-nucleosome、di-nucleosome 區段。如果只有單一寬峰 → 過度消化或染色質結構崩壞。

High-quality ATAC shows clear peaks at ~50, ~200, ~400 bp — nucleosome-free, mono-, and di-nucleosome fragments. A single broad peak indicates over-tagmentation or chromatin disruption.

scATAC

四、scATAC

10X scATAC 後常用兩個工具：

  • ArchR（Granja 2021）：速度與記憶體效能最佳、HDF5-based arrow files；建議 ≥50k cells 用 ArchR。
  • Signac（Stuart 2021）：與 Seurat 無縫整合，小 cohort 與 multi-modal 場景方便。

核心輸出：cell-by-peak matrix、TF deviation (chromVAR)、gene-activity score、co-accessibility (Cicero)、cell-type marker peaks。可與 scRNA WNN 整合（Ch 15）。

After 10X scATAC, two go-to tools:

  • ArchR (Granja 2021): best speed/RAM, HDF5-based arrow files; recommended for ≥50k cells.
  • Signac (Stuart 2021): seamless Seurat integration, convenient for small cohorts and multi-modal setups.

Core outputs: cell-by-peak matrix, TF deviation (chromVAR), gene-activity score, co-accessibility (Cicero), cell-type marker peaks. Can be integrated with scRNA via WNN (Ch 15).

程式碼

五、實作

# Bulk ATAC：bwa-mem2 → samtools → MACS3
bwa-mem2 mem -t 16 ref.fa s_R1.fq.gz s_R2.fq.gz | \
samtools sort -@ 8 -o s.bam -
samtools index s.bam

# 過濾 + Tn5 shift（deeptools alignmentSieve）
alignmentSieve -b s.bam -o s.shift.bam --ATACshift \
               --minMappingQuality 30 --blackListFileName blacklist.bed

# Peak calling（ATAC-specific）
macs3 callpeak -t s.shift.bam -f BAMPE -g hs \
            --nomodel --shift -75 --extsize 150 \
            -n sample --outdir peaks/ -q 0.05

# 差異分析（DESeq2 in R）
multiBamSummary BED-file --BED peaks/sample_peaks.narrowPeak \
                 --bamfiles c1.bam c2.bam t1.bam t2.bam -o counts.npz
# Signac — scATAC quickstart
library(Signac); library(Seurat)
counts <- Read10X_h5("filtered_peak_bc_matrix.h5")
meta   <- read.csv("singlecell.csv", row.names=1)

chrom <- CreateChromatinAssay(counts = counts, sep = c(":", "-"),
            fragments = "fragments.tsv.gz", genome = "hg38", min.cells = 10)
obj <- CreateSeuratObject(chrom, assay = "peaks", meta.data = meta)

obj <- RunTFIDF(obj) |> FindTopFeatures(min.cutoff="q5") |>
       RunSVD() |> RunUMAP(reduction="lsi", dims=2:30) |>
       FindNeighbors(reduction="lsi", dims=2:30) |>
       FindClusters()
DimPlot(obj, label=TRUE)
# ArchR — 大規模 scATAC（建議 ≥50k cells）
library(ArchR)
addArchRGenome("hg38")
ArrowFiles <- createArrowFiles(inputFiles = "fragments.tsv.gz",
                              sampleNames = "sample1", minTSS=4, minFrags=1000)
proj <- ArchRProject(ArrowFiles = ArrowFiles, outputDirectory="out")
proj <- addIterativeLSI(proj) |>
         addClusters(reducedDims="IterativeLSI") |>
         addUMAP(reducedDims="IterativeLSI") |>
         addGroupCoverages() |> addReproduciblePeakSet() |>
         addPeakMatrix() |> addMotifAnnotations(motifSet="cisbp")
plotEmbedding(proj, name="Clusters")
