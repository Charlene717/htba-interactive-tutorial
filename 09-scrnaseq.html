流程

一、12 步驟速覽

QC

nFeature、nCount、MT%、雙細胞偵測 (DoubletFinder / Scrublet)、ambient RNA (SoupX / CellBender)。

nFeature, nCount, MT%, doublet detection (DoubletFinder / Scrublet), ambient RNA (SoupX / CellBender).

Normalization

LogNormalize、SCTransform、scran pooling。SCTransform v2 (2022) 用 GLM-PCA 取代 NB regression，更穩定。

LogNormalize, SCTransform, scran pooling. SCTransform v2 (2022) uses GLM-PCA, more stable.

HVG → PCA → Neighbors → Clustering

高變異基因 → PCA 降維（多選 30-50 PC）→ KNN graph → Leiden（取代 Louvain，更穩定且支援 multi-resolution）。

HVGs → PCA (30–50 PCs) → KNN graph → Leiden (replaces Louvain — more stable, supports multi-resolution).

UMAP / Annotation

UMAP 2D 視覺化。註釋：marker-based (PanglaoDB、CellMarker)、reference-based (SingleR、Azimuth、Symphony)、LLM-assisted (GPTCelltype 2024)。

UMAP 2D viz. Annotation: marker-based (PanglaoDB, CellMarker), reference-based (SingleR, Azimuth, Symphony), LLM-assisted (GPTCelltype 2024).

DE / Integration / Cell-cell comm. / Trajectory

DE: Wilcoxon、MAST、edgeR/DESeq2 pseudo-bulk（推薦）。整合: Harmony、scVI、Seurat CCA、Scanorama（詳見 Ch 14）。Cell-cell: CellChat、CellPhoneDB、LIANA。軌跡: Monocle3、PAGA、scVelo。

DE: Wilcoxon, MAST, edgeR/DESeq2 pseudo-bulk (recommended). Integration: Harmony, scVI, Seurat CCA, Scanorama (Ch 14). Cell-cell: CellChat, CellPhoneDB, LIANA. Trajectory: Monocle3, PAGA, scVelo.

三、解析度與 cluster 數

Leiden / Louvain 的 resolution 越大，cluster 越多越細。沒有「正確答案」— 視生物問題與 marker 表現決定。實務上用 clustree 視覺化 resolution 之間的轉換。

Higher Leiden / Louvain resolution → more, finer clusters. No "right answer" — driven by biological question and marker expression. In practice use clustree to visualize transitions.

四、實作

# Seurat v5 全流程
library(Seurat); library(SeuratObject)
obj <- Read10X("filtered_feature_bc_matrix/") |>
       CreateSeuratObject(min.cells=3, min.features=200)
obj[["percent.mt"]] <- PercentageFeatureSet(obj, pattern="^MT-")
obj <- subset(obj, nFeature_RNA > 200 & nFeature_RNA < 6000 & percent.mt < 15)

obj <- SCTransform(obj) |>
       RunPCA(npcs=50) |>
       FindNeighbors(dims=1:30) |>
       FindClusters(resolution=0.5, algorithm=4) |>   # 4 = Leiden
       RunUMAP(dims=1:30)

# Marker + Annotation（Azimuth reference-based）
markers <- FindAllMarkers(obj, only.pos=TRUE, min.pct=0.25)
library(Azimuth)
obj <- RunAzimuth(obj, reference="pbmcref")
DimPlot(obj, group.by="predicted.celltype.l2", label=TRUE)
# Scanpy 全流程
import scanpy as sc
adata = sc.read_10x_mtx("filtered_feature_bc_matrix/")
sc.pp.filter_cells(adata, min_genes=200)
sc.pp.filter_genes(adata, min_cells=3)
adata.var["mt"] = adata.var_names.str.startswith("MT-")
sc.pp.calculate_qc_metrics(adata, qc_vars=["mt"], inplace=True)
adata = adata[(adata.obs.n_genes_by_counts < 6000) &
              (adata.obs.pct_counts_mt < 15)]

sc.pp.normalize_total(adata, target_sum=1e4)
sc.pp.log1p(adata)
sc.pp.highly_variable_genes(adata, n_top_genes=3000, flavor="seurat_v3")
sc.pp.scale(adata); sc.tl.pca(adata, n_comps=50)
sc.pp.neighbors(adata, n_pcs=30)
sc.tl.leiden(adata, resolution=0.5)
sc.tl.umap(adata)
sc.pl.umap(adata, color=["leiden"])

# Annotation：CellTypist（人類 ~25 個 reference）
import celltypist
preds = celltypist.annotate(adata, model="Immune_All_Low.pkl",
                            majority_voting=True)
