Bulk RNA-seq 差異表達

三大主流工具 (DESeq2 / edgeR / limma-voom)、shrinkage、normalize 與 MA/volcano plot — 一條龍打通。

The three mainstream tools (DESeq2 / edgeR / limma-voom), shrinkage, normalization, and MA/volcano plots — end-to-end.

輸入

一、counts 矩陣

Bulk RNA-seq 差異分析的起點是 gene × sample 整數 count matrix。三種主要產出路徑：

  • STAR + featureCounts / HTSeq：傳統路徑，產 gene-level counts。
  • Salmon / kallisto → tximport：transcript-level → 用 tximport(..., countsFromAbundance="lengthScaledTPM") 聚合到 gene-level，自動處理 isoform length bias。
  • nf-core/rnaseq：上述兩條路徑皆內建，建議直接用。

Bulk DE starts from a gene × sample integer count matrix. Three main routes:

  • STAR + featureCounts / HTSeq: classic gene-level counts.
  • Salmon / kallisto → tximport: transcript-level → aggregate to gene with countsFromAbundance="lengthScaledTPM", auto-correcting isoform length bias.
  • nf-core/rnaseq: both routes built-in — preferred.
⚠️
關鍵：差異分析務必用 raw counts，不要用 TPM / FPKM / RPKM。原因：DE 工具自己會在內部模型 dispersion 與 size factor，給標準化後值會破壞變異結構。 Critical: Use raw counts for DE — never TPM / FPKM / RPKM. DE tools model dispersion and size factors internally; passing normalized values destroys the variance structure.

💡
實務建議：n < 50 用 DESeq2；n > 100 或 cohort study 用 limma-voom（速度差距明顯）；複雜 multi-factor design 或想精細控制 dispersion 用 edgeR。三者結果在合理 design 下高度一致 — 工具差異 ≪ 設計差異。 Practical: Use DESeq2 for n < 50; limma-voom for n > 100 or cohorts (much faster); edgeR for complex multi-factor designs or fine dispersion control. Under sound designs the three agree closely — tool choice ≪ design choice.

四、實作

# DESeq2 — 工業標準
library(DESeq2); library(tximport)
txi  <- tximport(salmon_files, type="salmon", tx2gene=tx2gene)
dds  <- DESeqDataSetFromTximport(txi, colData=coldata, design=~batch+condition)
dds  <- DESeq(dds)
res  <- results(dds, contrast=c("condition","treated","control"))

# Shrinkage（推薦做圖前）
res_sh <- lfcShrink(dds, coef="condition_treated_vs_control", type="apeglm")

# 視覺化
plotMA(res_sh, ylim=c(-4,4))
summary(res, alpha=0.05)
library(EnhancedVolcano)
EnhancedVolcano(res_sh, lab=rownames(res_sh),
                x="log2FoldChange", y="padj",
                pCutoff=0.05, FCcutoff=1)
# edgeR — GLM + QLF
library(edgeR)
y <- DGEList(counts=counts, group=group)
keep <- filterByExpr(y, group=group); y <- y[keep, ]
y <- calcNormFactors(y, method="TMM")

design <- model.matrix(~batch+condition)
y <- estimateDisp(y, design)
fit <- glmQLFit(y, design)
qlf <- glmQLFTest(fit, coef="conditiontreated")
topTags(qlf, n=20)
# limma-voom — 大 cohort 首選
library(limma); library(edgeR)
y <- DGEList(counts) |> calcNormFactors()
design <- model.matrix(~batch+condition)
v <- voom(y, design, plot=TRUE)
fit <- lmFit(v, design) |> eBayes()
topTable(fit, coef="conditiontreated", n=20)
# pyDESeq2 — Python 移植版（2023+）
# pip install pydeseq2
from pydeseq2.dds import DeseqDataSet
from pydeseq2.ds  import DeseqStats

dds = DeseqDataSet(counts=counts_df, metadata=meta_df,
                   design_factors=["batch", "condition"])
dds.deseq2()
ds = DeseqStats(dds, contrast=["condition", "treated", "control"])
ds.summary()
res_df = ds.results_df.sort_values("padj")
