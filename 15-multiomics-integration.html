CH 15 / 20 · BLOCK 5 INTEGRATION

多體學垂直整合

MOFA+、Seurat WNN、MultiVI、totalVI、GLUE、DIABLO — 同樣本跨組學整合方法總覽。

MOFA+, Seurat WNN, MultiVI, totalVI, GLUE, DIABLO — same-sample cross-omics integration methods.

兩種維度

一、水平 vs. 垂直

↔️ 水平整合

同 omics × 跨樣本（Ch 14）。「拼接 cohort」。

Same omics × across samples (Ch 14). "Stitch cohorts together."

↕️ 垂直整合

同樣本 × 跨 omics（本章）。需要每個樣本都做了 ≥ 2 種 omics（如 RNA + ATAC + Protein），不然只能用 anchor-based 對齊。

Same sample × across omics (this chapter). Requires ≥ 2 omics per sample (e.g., RNA + ATAC + Protein); otherwise anchor-based alignment is needed.

方法家族

二、四大方法家族

🧮 因子模型

找出多 omics 共享的 latent factor，每 factor 解釋多少變異來自哪 omics。

代表：MOFA / MOFA+ / MEFISTO（時間軸）、JIVE、iCluster+、DIABLO（mixOmics）。

Identify latent factors shared across omics; each factor explains variance from which omics layers.

Examples: MOFA / MOFA+ / MEFISTO (temporal), JIVE, iCluster+, DIABLO (mixOmics).

🤖 深度生成模型

每 omics 一個 encoder，共用 latent space。

代表：totalVI（RNA+ADT, CITE-seq）、MultiVI（RNA+ATAC, 10x Multiome）、Multigrate、UnitedNet、scMM。

One encoder per omics, shared latent space.

Examples: totalVI (RNA+ADT, CITE-seq), MultiVI (RNA+ATAC, 10x Multiome), Multigrate, UnitedNet, scMM.

🕸️ 圖學習 / 配對

用 graph attention / regulatory prior 跨 omics 對齊。

代表：WNN（Seurat v4, 加權最近鄰）、GLUE（scglue, RNA+ATAC graph-linked）、UINMF。

Graph attention or regulatory priors align across omics.

Examples: WNN (Seurat v4, weighted nearest neighbors), GLUE (scglue, RNA+ATAC graph-linked), UINMF.

🎯 矩陣分解

NMF / SVD 跨 omics 共同分解。

代表：LIGER（iNMF）、scAI、CoGAPS。

NMF / SVD jointly decompose across omics.

Examples: LIGER (iNMF), scAI, CoGAPS.

💡
2024 Nature Methods benchmark：Seurat WNN、Multigrate、Matilda、UnitedNet 在多 cohort 上表現好；GLUE 在 scRNA+scATAC 對齊上勝出；MOFA+ 在概念上最直觀但記憶體大、不擴展。MOFA+ 在 bulk multi-omics 仍是最常用的入門工具。 2024 Nature Methods benchmark: Seurat WNN, Multigrate, Matilda, UnitedNet perform well across cohorts; GLUE wins on scRNA + scATAC alignment; MOFA+ is the most interpretable but memory-heavy and doesn't scale. MOFA+ remains the most popular entry point for bulk multi-omics.
選型決策樹

三、選型決策

🌳 按資料形態挑工具

Q1
每個樣本都做 ≥ 2 種 omics（matched）？ → 否 → 用 GLUE / Tangram / Conos 做 unmatched 對齊；→ 是 → 繼續 Q2。
Q2
Bulk 還是 single-cell？ Bulk → MOFA+（首選） / DIABLO / iCluster+；Single-cell → 繼續 Q3。
Q3
CITE-seq (RNA+ADT)？ → totalVI / Seurat WNN。10x Multiome (RNA+ATAC)？ → MultiVI / Seurat WNN / GLUE。
Q4
規模 > 500k cells？ → 優先用 scVI 家族（VAE 容易 scale）。
Q1
Are ≥ 2 omics measured on each sample (matched)? → No → use GLUE / Tangram / Conos for unmatched alignment; → Yes → go to Q2.
Q2
Bulk or single-cell? Bulk → MOFA+ (first choice) / DIABLO / iCluster+; Single-cell → Q3.
Q3
CITE-seq (RNA+ADT)? → totalVI / Seurat WNN. 10x Multiome (RNA+ATAC)? → MultiVI / Seurat WNN / GLUE.
Q4
Scale > 500k cells? → prefer the scVI family (VAEs scale well).
互動

四、MOFA+ 因子變異

MOFA+ 輸出每個 factor 在每個 omics 解釋的變異比例。某些 factor 為「共享 (shared)」、某些只在單一 omics 強。

MOFA+ outputs the fraction of variance explained by each factor in each omics. Some factors are "shared," others active in a single omics.

程式碼

五、實作

# MOFA+ — 多 omics 因子分析
library(MOFA2)
data <- list(RNA=rna_matrix, ATAC=atac_matrix, Methyl=methyl_matrix)
mofa <- create_mofa(data)

mod_opts <- get_default_model_options(mofa)
mod_opts$num_factors <- 15
mofa <- prepare_mofa(mofa, model_options=mod_opts)

mofa <- run_mofa(mofa, outfile="mofa_model.hdf5")

# 解釋每 factor 變異
plot_variance_explained(mofa, x="view", y="factor")
factors <- get_factors(mofa)[[1]]
# 把 factor 當 PC 一樣下游分群、做 GSEA
# Seurat WNN — RNA + ADT (CITE-seq) 或 RNA + ATAC
library(Seurat)
obj <- CreateSeuratObject(rna) |> SCTransform() |>
        RunPCA(reduction.name="pca_rna")
obj[["ADT"]] <- CreateAssayObject(adt)
DefaultAssay(obj) <- "ADT"
obj <- NormalizeData(obj, normalization.method="CLR", margin=2) |>
        ScaleData() |> RunPCA(reduction.name="pca_adt", npcs=20)

obj <- FindMultiModalNeighbors(obj,
         reduction.list = list("pca_rna", "pca_adt"),
         dims.list = list(1:30, 1:20))
obj <- RunUMAP(obj, nn.name="weighted.nn", reduction.name="wnn.umap")
obj <- FindClusters(obj, graph.name="wsnn", algorithm=4)
# MultiVI — 10x Multiome (RNA + ATAC) integration
import scvi, anndata as ad
adata = ad.concat([adata_rna, adata_atac], axis=1, join="outer")
adata.var["modality"] = ["RNA"]*adata_rna.shape[1] + ["ATAC"]*adata_atac.shape[1]

scvi.model.MULTIVI.setup_anndata(adata, batch_key="sample")
m = scvi.model.MULTIVI(adata,
       n_genes=(adata.var["modality"]=="RNA").sum(),
       n_regions=(adata.var["modality"]=="ATAC").sum())
m.train()
adata.obsm["X_multivi"] = m.get_latent_representation()
# GLUE — graph-linked unified embedding（unmatched 也可）
import scglue, networkx as nx
# 建立 prior knowledge graph（RNA-ATAC link 由 peak-gene proximity）
guidance = scglue.genomics.rna_anchored_guidance_graph(rna_ad, atac_ad)

scglue.models.configure_dataset(rna_ad, "NB", use_highly_variable=True, use_layer="counts")
scglue.models.configure_dataset(atac_ad, "NB", use_highly_variable=True, use_rep="X_lsi")
glue = scglue.models.fit_SCGLUE({"rna":rna_ad, "atac":atac_ad}, guidance)
rna_ad.obsm["X_glue"]  = glue.encode_data("rna", rna_ad)
atac_ad.obsm["X_glue"] = glue.encode_data("atac", atac_ad)
