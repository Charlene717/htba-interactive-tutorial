可重現性、雲端與倫理

Snakemake / Nextflow、Singularity、dbGaP / EGA、IRB、ELSI、EU AI Act — 把研究做得能被別人重做、用得安全。

Snakemake / Nextflow, Singularity, dbGaP / EGA, IRB, ELSI, EU AI Act — make your research reproducible and ethically deployable.

可重現性

一、五層堆疊

版本控制

Git + GitHub / GitLab；包括 code、metadata、analysis notes。大檔案用 git-lfs / DVC / OSF。

Git + GitHub / GitLab; covers code, metadata, analysis notes. Large files via git-lfs / DVC / OSF.

環境

Conda / Mamba / Pixi（Python+R 套件）、renv（R 專屬 lockfile）、uv（Python 快速 venv）、Bioconductor 版本鎖。

Conda / Mamba / Pixi (Python+R), renv (R lockfile), uv (fast Python venv), Bioconductor version lock.

容器

Docker（本地 / 雲）、Singularity / Apptainer（HPC 與 root-less）、BioContainers 提供 Bioconda 套件對應 image。

Docker (local/cloud), Singularity / Apptainer (HPC, root-less), BioContainers for Bioconda-equivalent images.

管線

Nextflow（DSL2 + nf-core）、Snakemake（Python DSL）、Cromwell + WDL（Broad / Terra）、Galaxy（GUI）。

Nextflow (DSL2 + nf-core), Snakemake (Python DSL), Cromwell + WDL (Broad / Terra), Galaxy (GUI).

報告

Quarto（R / Python / Julia 通用）、R Markdown、Jupyter、MultiQC（QC 報告整合）。

Quarto (R/Python/Julia), R Markdown, Jupyter, MultiQC (QC aggregation).

雲端

二、雲端與 HPC

☁️ 公雲生醫平台

Terra（Broad）、DNAnexus、AnVIL（NHGRI）、Seven Bridges CGC、All of Us Researcher Workbench。資料受控存取在雲端不下載到本機。

🖥️ HPC 排程

Slurm（最普遍）、SGE / PBS / LSF。Nextflow / Snakemake 都有 executor 直接送 job。

🚀 GPU 訓練

RTX 6000 Ada / H100 / B200；用 PyTorch Lightning、NVIDIA Parabricks（GPU GATK）、RAPIDS（cuDF / cuML）加速。

📡 資料 streaming

ONT MinKNOW 即時 basecalling、Illumina BaseSpace、AWS HealthOmics 物件儲存。

💰 成本管理

spot instance、auto-scaling、cold storage（Glacier）、egress 費用最易暴衝。先做 dry-run 試算。

🔐 安全

HIPAA、GDPR、TWN PIPA。受控存取用 dbGaP / EGA token；雲端啟用 KMS encryption + VPC private subnet。

倫理

三、倫理與隱私

IRB / 倫委會：任何涉及人類受試者的高通量資料都需先過 IRB；二次分析 deidentified 公開資料常可申請 exemption。

知情同意：NIH 自 2024 起 Data Management & Sharing Plan 為強制要求；同意書要含「未來其他用途」與「可否撤回」。

再識別風險：基因組 + 鄰近家族成員 ≥ 30 SNPs 就可能被 re-identify。差分隱私 (DP) 與安全多方計算 (SMC / TEE) 是當前趨勢。

EU AI Act（2024/1689，2024 通過、2026 全面適用）：把醫療診斷 AI 歸為 high-risk，需做 risk management、data governance、human oversight、post-market monitoring。FDA 自 2023 也有 PCCP (Predetermined Change Control Plan) 規範持續學習模型。

公平性：Obermeyer et al. 2019 Science 顯示醫療演算法可能複製族裔偏見；Bias audit、CONSORT-AI / SPIRIT-AI 報告準則是必要實踐。

IRB: Human-subject high-throughput data needs IRB approval upfront; secondary analyses of deidentified public data often qualify for exemption.

Informed consent: NIH Data Management & Sharing Plan has been mandatory since 2024; consent must cover "future other uses" and "withdrawal."

Re-identification risk: Genome + close-relative pedigree at ≥30 SNPs can re-identify subjects. Differential privacy (DP), secure multiparty computation (SMC / TEE) are trending defenses.

EU AI Act (2024/1689): diagnostic AI is high-risk — requires risk management, data governance, human oversight, post-market monitoring. FDA's PCCP (since 2023) governs continuously learning models.

Fairness: Obermeyer et al. (Science 2019) showed clinical algorithms can replicate ethnic bias; bias audits and CONSORT-AI / SPIRIT-AI reporting standards are essential.

五、實作

# nf-core/rnaseq 在 HPC + Singularity
nextflow run nf-core/rnaseq -r 3.16.0 \
        --input samplesheet.csv --outdir results/ \
        --genome GRCh38 --aligner star_salmon \
        -profile singularity,slurm \
        -resume

# 在 AWS HealthOmics 或 Terra 上執行（節省成本）
nextflow run nf-core/rnaseq -profile awsbatch \
        --awsregion us-east-1 --awsqueue arn:aws:batch:...
# Snakemake — 強型別 + auto-deploy conda env
# config.yaml + Snakefile
configfile: "config.yaml"

rule all:
    input: expand("qc/{s}.html", s=config["samples"])

rule fastqc:
    input: "raw/{s}.fq.gz"
    output: "qc/{s}.html"
    conda: "envs/fastqc.yaml"   # 自動建環境
    container: "docker://biocontainers/fastqc:v0.12.1"
    resources: mem_mb=4000, runtime=30
    shell: "fastqc -o qc/ {input}"

# 執行
# snakemake --use-conda --cores 16
# snakemake --use-singularity --slurm
# Apptainer — root-less HPC 友善
apptainer build samtools.sif docker://staphb/samtools:1.20

# 執行
apptainer exec --bind /data:/data samtools.sif \
          samtools view -b -q 30 input.bam > out.bam

# Pixi — 跨平台 conda + lockfile
pixi init my-analysis
cd my-analysis
pixi add -c bioconda samtools=1.20 bcftools=1.21 nextflow
pixi run snakemake --cores 8

# 自動產生 pixi.lock，git commit 即可跨機器還原
