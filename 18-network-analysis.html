網絡分析與圖學習

WGCNA、STRING、Cytoscape、SCENIC、graph neural networks — 從共表達到 GNN 預測。

WGCNA, STRING, Cytoscape, SCENIC, graph neural networks — from co-expression to GNN prediction.

網絡類型

一、四大網絡

🧬 共表達

節點=基因；邊=表達相關。WGCNA（Langfelder 2008）是經典；現代替代：hdWGCNA（single-cell）、CEMiTool、fastBMA。輸出 module + hub genes。

Nodes = genes; edges = expression correlation. WGCNA (Langfelder 2008) is classic; modern: hdWGCNA (single-cell), CEMiTool, fastBMA. Output: modules + hub genes.

🤝 PPI 蛋白互動

節點=蛋白；邊=已知 / 預測互作。資料庫：STRING、BioGRID、IntAct、HuRI。分析：propagation、module detection (Louvain)、visualization (Cytoscape)。

Nodes = proteins; edges = known/predicted interactions. DBs: STRING, BioGRID, IntAct, HuRI. Analysis: propagation, module detection (Louvain), visualization (Cytoscape).

🎛️ 調控網絡

節點=基因 / TF；邊=調控關係。SCENIC / SCENIC+（scRNA + ATAC + motif）、GENIE3、CellOracle（含 KO 模擬）、Pando。

Nodes = genes/TFs; edges = regulation. SCENIC / SCENIC+ (scRNA + ATAC + motif), GENIE3, CellOracle (with KO simulation), Pando.

🌐 知識圖譜

異質節點（基因 / 化合物 / 疾病 / 表型）+ 關係邊。OpenTargets Platform、Hetionet、PrimeKG、BioBricks；用 graph embedding (TransE, GAT) 做 drug repurposing。

Heterogeneous nodes (genes / compounds / diseases / phenotypes) + relation edges. OpenTargets, Hetionet, PrimeKG, BioBricks; graph embeddings (TransE, GAT) for drug repurposing.

三、scale-free 拓樸

WGCNA 透過 soft-thresholding power 把相關矩陣轉成「scale-free」（少數 hub + 多數低度節點）。R² > 0.8 通常為 reasonable power。

WGCNA uses a soft-thresholding power to turn correlation into "scale-free" topology (few hubs + many low-degree nodes). R² > 0.8 typically marks a reasonable power.

四、實作

# WGCNA — 共表達 module
library(WGCNA); options(stringsAsFactors=FALSE)
enableWGCNAThreads()

datExpr <- t(log2(counts+1))      # sample × gene
sft <- pickSoftThreshold(datExpr, powerVector = 1:20)
plot(sft$fitIndices$Power, -sign(sft$fitIndices$slope)*sft$fitIndices$SFT.R.sq)

net <- blockwiseModules(datExpr, power = 6,
              TOMType = "signed", minModuleSize = 30,
              mergeCutHeight = 0.25, numericLabels = TRUE)

# Module trait correlation
ME <- net$MEs
mod_trait <- cor(ME, pheno_traits, use="p")
labeledHeatmap(mod_trait, xLabels=colnames(pheno_traits),
                yLabels=colnames(ME))
# STRINGdb — PPI 與 module enrichment
library(STRINGdb)
sdb <- STRINGdb$new(version="12.0", species=9606,
                       score_threshold=700, input_directory="")
mapped <- sdb$map(sig_genes, "gene", removeUnmappedRows=TRUE)
sdb$plot_network(mapped$STRING_id)

enrich <- sdb$get_enrichment(mapped$STRING_id, category="KEGG")
head(enrich)
# pySCENIC — TF regulon discovery（scRNA）
pyscenic grn expr.tsv tfs.txt -o adjacencies.tsv --num_workers 16
pyscenic ctx adjacencies.tsv \
         hg38_motif_cb.feather --annotations_fname motifs.tbl \
         --expression_mtx_fname expr.tsv -o regulons.csv --num_workers 16
pyscenic aucell expr.tsv regulons.csv -o aucell.csv

# GNN (PyG): node2vec on STRING graph
import torch
from torch_geometric.datasets import AttributedGraphDataset
from torch_geometric.nn import Node2Vec
model = Node2Vec(edge_index, embedding_dim=128, walk_length=20,
                 context_size=10).to("cuda")
opt = torch.optim.SparseAdam(model.parameters(), lr=0.01)
for epoch in range(50):
    loss = model.loss(model.sample(...)); loss.backward(); opt.step()
