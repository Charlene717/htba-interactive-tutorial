蛋白質體

DDA vs. DIA、MaxQuant / FragPipe / DIA-NN、PTM、CITE-seq、Olink、SomaScan — 從 MS 到單細胞蛋白。

DDA vs. DIA, MaxQuant / FragPipe / DIA-NN, PTMs, CITE-seq, Olink, SomaScan — from MS to single-cell proteins.

兩大模式

一、DDA vs. DIA

🎯 DDA (Data-Dependent Acquisition)

每個 MS1 scan 後挑前 N 個高豐度 precursor 做 MS2 fragmentation。經典模式，低豐度蛋白容易漏。工具：MaxQuant、Proteome Discoverer、FragPipe（含 MSFragger，速度最快）。

For each MS1 scan, pick top-N abundant precursors for MS2 fragmentation. Classic mode but low-abundance proteins are often missed. Tools: MaxQuant, Proteome Discoverer, FragPipe (MSFragger inside — fastest).

🌊 DIA (Data-Independent Acquisition)

把 m/z 範圍切成固定 windows，每 window 內所有 ion 都做 fragmentation。無偏取樣、reproducible、適合大 cohort。工具：DIA-NN（library-free 主流）、Spectronaut、EncyclopeDIA。深度學習預測 spectral library（Prosit、DeepRT）大幅提升 ID 數。

Divide m/z range into fixed windows; fragment all ions in each window. Unbiased sampling, reproducible, ideal for large cohorts. Tools: DIA-NN (the library-free workhorse), Spectronaut, EncyclopeDIA. DL-predicted spectral libraries (Prosit, DeepRT) boost IDs substantially.

💡
2024-2025 趨勢：DIA 已是 cohort proteomics 主流；DIA-NN 1.9 + AlphaPept / AlphaPeptDeep 把 library-free 流程推到 >10k protein IDs / plasma sample / 半小時 gradient。 2024-2025 trend: DIA dominates cohort proteomics; DIA-NN 1.9 + AlphaPept / AlphaPeptDeep push library-free workflows to >10k protein IDs per plasma sample in 30 min gradients.
流程

二、分析流程

Raw → mzML

原廠 .raw / .wiff → 用 ProteoWizard msconvert 或 ThermoRawFileParser 轉開放標準 .mzML。

Vendor .raw / .wiff → use ProteoWizard msconvert or ThermoRawFileParser for open .mzML.

搜尋 / 量化

DDA：MaxQuant LFQ / FragPipe LFQ / TMT-MS3；DIA：DIA-NN library-free 或 spectral lib + Prosit。

DDA: MaxQuant LFQ / FragPipe LFQ / TMT-MS3; DIA: DIA-NN library-free or spectral lib + Prosit.

FDR 控制

PSM-level、peptide-level、protein-level 三層 FDR；常用 1% PSM FDR + 1% protein FDR。Picked group FDR / Pyteomics 模組更嚴格。

PSM-, peptide-, protein-level FDR; common 1% PSM + 1% protein FDR. Picked group FDR / Pyteomics modules are stricter.

統計

MSstats、limma、ProlfqUMI / MSqRob2。missing value 處理：MNAR + MAR 混合，常用 perseus 的 imputation 或 MissForest。

MSstats, limma, ProlfqUMI / MSqRob2. Missing values: mixed MNAR + MAR — use perseus imputation or MissForest.

PTM

磷酸化、乙醯化、glycosylation：open search（MSFragger / Crux）、Skyline。site-level localization 用 PTM-Shepherd、AScore。

Phosphorylation, acetylation, glycosylation: open search (MSFragger / Crux), Skyline. Site localization: PTM-Shepherd, AScore.

替代平台

三、affinity-based 平台

🦠 CITE-seq

scRNA + ~100 抗體 oligo barcode。Stoeckius 2017。同細胞測 mRNA + 蛋白表面 marker。已是免疫 atlas 標配。

scRNA + ~100 oligo-barcoded antibodies (Stoeckius 2017). mRNA + surface protein per cell. Standard for immune atlases.

🎯 Olink

PEA（proximity extension assay）：每 protein 2 個抗體靠近時產生擴增信號。3072-plex、~1 µL 血漿、用於 UK Biobank PPP。

Proximity extension assay: two antibodies per protein generate signal when close. 3072-plex, ~1 µL plasma; used by UK Biobank PPP.

🧬 SomaScan

SOMAmer（modified aptamer）測 ~11k proteins，血漿 / 血清。常用於 plasma proteome、aging clock。

SOMAmers (modified aptamers) measure ~11k proteins in plasma/serum. Common for plasma proteome, aging clocks.

scProteomics MS

SCoPE2 / SCoPE-MS2、nPOP、plexDIA：單一 HeLa 細胞已可量化 ~3000 蛋白；Thermo Stellar MS 2024 為硬體里程碑。

SCoPE2 / SCoPE-MS2, nPOP, plexDIA: ~3000 proteins per HeLa cell; Thermo Stellar MS (2024) is a hardware milestone.

📦 PRIDE / MassIVE

PRIDE（EBI）是 MS proteomics 主流公開資料庫；MassIVE 為 UCSD/UCSD CCMS。提交需 .raw + 結果 + protocol。

PRIDE (EBI) is the main public MS proteomics repository; MassIVE is UCSD/CCMS. Submission requires .raw + results + protocol.

🏆 PCAWG → CPTAC

CPTAC（NCI）整合癌症 proteogenomics（DNA + RNA + protein + phospho + PTM）。是 multi-omics 整合最完整的公開資料。

CPTAC (NCI) integrates cancer proteogenomics (DNA + RNA + protein + phospho + PTM). The most complete public multi-omics dataset.

# DIA-NN library-free 模式（含 in silico spectral lib）
diann --f sample1.mzML --f sample2.mzML --f sample3.mzML \
      --fasta human_uniprot.fa \
      --predictor --threads 16 --qvalue 0.01 \
      --matrices --report-lib-info \
      --out report.tsv --out-lib in_silico_lib.tsv

# 結果讀入 R 做後續統計
# MSstats — 蛋白層級統計
library(MSstats)
raw <- DIANNtoMSstatsFormat(read.delim("report.tsv"),
                              annotation = sample_annot)
proc <- dataProcess(raw, normalization = "equalizeMedians",
                      summaryMethod = "TMP", censoredInt = "NA")

cm  <- matrix(c(-1, 1), nrow=1, dimnames=list("TvsC", c("Ctrl","Treated")))
res <- groupComparison(contrast.matrix = cm, data = proc)$ComparisonResult

# Volcano
library(ggplot2)
ggplot(res, aes(log2FC, -log10(adj.pvalue))) + geom_point()
