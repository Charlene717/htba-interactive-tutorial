長讀全長轉錄組

PacBio Iso-Seq + ONT direct-RNA：完整 isoform、新轉錄本與 m6A 同步偵測。

PacBio Iso-Seq + ONT direct-RNA: complete isoforms, novel transcripts, and joint m6A detection.

技術

一、為什麼需要長讀？

短讀（150 bp）需要重新組裝 isoform，常會把多個 transcript 的 reads 重新拼接，造成 isoform-level 估算偏差。長讀（>1 kb，常見全長 1-10 kb）能一條 read 涵蓋整條 mRNA，直接觀察 splice 組合與 UTR；對新 isoform、融合轉錄本、allelic-specific isoform、A-to-I editing、m6A modification都更有信心。

2024-2025 進展：PacBio Revio 通量提升 15×；ONT R10.4.1 RNA-002 化學 accuracy ~98%；scIso-Seq 與 ONT scNanopore-seq 已可做單細胞 isoform。

Short reads (150 bp) reassemble isoforms, often mixing reads across transcripts and biasing isoform-level estimates. Long reads (>1 kb, often 1–10 kb full-length) can span the whole mRNA in one read, observing splice combinations and UTRs directly. Better for novel isoforms, fusion transcripts, allelic-specific isoforms, A-to-I editing, m6A modification.

2024-2025: PacBio Revio raised throughput 15×; ONT R10.4.1 RNA-002 chemistry hits ~98% accuracy; scIso-Seq and ONT scNanopore-seq enable single-cell isoform analysis.

互動

四、Isoform 解析力

調整讀長與基因 isoform 數，估算「完整 isoform 偵測率」（簡化模型）。長讀讀越長越能整段 cover；短讀則需要做組裝拼接。

Move read length and isoform count to estimate "complete isoform detection rate" (simplified model). Longer reads cover the full transcript; short reads need re-assembly.

五、實作

# PacBio Iso-Seq full pipeline
ccs subreads.bam ccs.bam --min-rq 0.9
lima ccs.bam primers.fa demux.bam --isoseq --peek-guess
isoseq3 refine demux.bam primers.fa flnc.bam --require-polya
isoseq3 cluster flnc.bam clustered.bam

# 比對到基因組
minimap2 -ax splice -uf --secondary=no -C5 -t 16 \
         ref.fa clustered.hq.fasta | samtools sort -o aln.bam

# 量化 + isoform discovery（Bambu）
# 在 R 中做：詳見下一個 tab
# Bambu — context-aware isoform discovery + quantification
library(bambu)
test <- bambu(reads = c("s1.bam", "s2.bam", "s3.bam"),
               annotations = prepareAnnotations("gencode.gtf"),
               genome = "ref.fa", ncore = 8)

# 萃取結果
counts <- assays(test)$counts
writeBambuOutput(test, path = "bambu_output/")

# 下游差異 isoform 用 DRIMSeq / satuRn
library(satuRn)
sce  <- fitDTU(test, formula = ~condition)
res  <- testDTU(sce, contrasts = c("trtvsctrl"))
# m6Anet — ONT direct-RNA m6A detection
# pip install m6anet
m6anet-dataprep --eventalign eventalign.txt \
                --out_dir dataprep_out --n_processes 8
m6anet-run_inference --input_dir dataprep_out \
                     --out_dir inference_out --infer_mod_rate

# Python: 篩選高信賴 m6A 位點
import pandas as pd
df = pd.read_csv("inference_out/data.site_proba.csv")
m6a = df[(df.probability_modified > 0.9) & (df.mod_ratio > 0.5)]
print(f"High-confidence m6A sites: {len(m6a)}")
