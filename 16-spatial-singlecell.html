CH 16 / 20 · BLOCK 5 INTEGRATION

空間 × 單細胞整合

把 scRNA 的細胞型態地圖「回貼」到空間切片：Tangram、cell2location、SPOTlight、RCTD、Stereoscope、CellTrek。

Map scRNA cell-type labels back onto spatial sections: Tangram, cell2location, SPOTlight, RCTD, Stereoscope, CellTrek.

動機

一、整合動機

Visium / Stereo-seq spot（55 µm / 8-16 µm bin）覆蓋多個細胞，無法區分 cell type；而 scRNA-seq 有清楚細胞身份卻失去空間。整合兩者可獲得「空間化的細胞型態地圖」與「細胞型態解析的差異表達」。Xenium / MERSCOPE 雖達單細胞解析度，但只測 panel-level，需要與 scRNA 整合才能擴展到全轉錄組。

Visium / Stereo-seq spots (55 µm / 8–16 µm bins) cover multiple cells with unknown identities; scRNA-seq has clear identities but no spatial context. Integration yields spatially resolved cell-type maps and cell-type-resolved DE in space. Xenium / MERSCOPE achieve single-cell resolution but only on panels — must integrate with scRNA for whole-transcriptome inference.

互動

三、Spot deconvolution

每個 spot 是不同細胞型態的混合。深度學習方法（cell2location）vs. 線性方法（SPOTlight）vs. 直接 mapping（Tangram）行為不同。

Each spot is a cell-type mixture. DL methods (cell2location) vs. linear (SPOTlight) vs. direct mapping (Tangram) behave differently.

四、實作

# cell2location — 兩階段
import cell2location
from cell2location.models import RegressionModel, Cell2location

# 1. 從 scRNA 學 cell-type 參考
RegressionModel.setup_anndata(adata_sc, batch_key="sample", labels_key="cell_type")
ref = RegressionModel(adata_sc); ref.train(max_epochs=250)
inf_aver = ref.export_posterior(adata_sc)["means_per_cluster_mu_fg"]

# 2. 在 Visium 上反算 abundance
Cell2location.setup_anndata(adata_vis, batch_key="sample")
mod = Cell2location(adata_vis, cell_state_df=inf_aver,
                    N_cells_per_location=8, detection_alpha=20)
mod.train(max_epochs=30000)
adata_vis = mod.export_posterior(adata_vis)
# Tangram — 把 sc 細胞貼到 spatial 座標
import tangram as tg
tg.pp_adatas(adata_sc, adata_sp, genes=marker_genes)

ad_map = tg.map_cells_to_space(adata_sc, adata_sp,
                                  mode="cells", density_prior="rna_count_based",
                                  num_epochs=1000, device="cuda:0")

# 把 cell-type 投影到 spatial
tg.project_cell_annotations(ad_map, adata_sp, annotation="cell_type")
tg.plot_cell_annotation_sc(adata_sp, ["T cell","B cell","Macrophage"],
                              x="x", y="y", perc=0.02)
# RCTD (spacexr) — robust doublet-aware deconvolution
library(spacexr); library(Matrix)

ref <- Reference(counts_sc, cell_types=sc_meta$ct,
                 nUMI=colSums(counts_sc))
puck <- SpatialRNA(coords=sp_coords, counts=counts_sp,
                   nUMI=colSums(counts_sp))

myRCTD <- create.RCTD(puck, ref, max_cores=8)
myRCTD <- run.RCTD(myRCTD, doublet_mode="doublet")
results <- myRCTD@results$results_df
head(results)
