DNA 甲基化

Bisulfite、EM-seq、EPIC array、長讀直讀 — 4 種測量加上 DMP/DMR 統計與 epigenetic age clock。

Bisulfite, EM-seq, EPIC arrays, long-read direct calling — four measurements plus DMP/DMR statistics and epigenetic age clocks.

技術

一、四種測量技術

技術原理覆蓋優缺點
WGBSBisulfite → C→U（除非 5mC）全 CpG破壞、高深度
RRBSMspI 富集~5%便宜、限制
EM-seq酵素全 CpG推薦取代 WGBS
EPIC v2 array~935k 位點固定最便宜、固定
ONT / PacBio direct所有修飾完整2024+ 主流
分析流程

二、分析流程

比對

Bismark（最通用）、bwa-meth（快）、BS-Seeker2。EM-seq 用同樣工具。

Bismark (most popular), bwa-meth (fast), BS-Seeker2. Same tools for EM-seq.

甲基化呼叫

Bismark methylation extractor 輸出 per-CpG β value = methylated / (methylated + unmethylated)。

Bismark methylation extractor outputs per-CpG β = methylated / (methylated + unmethylated).

DMP / DMR

methylKit、DSS（Bayesian smoothing）、bsseq、RnBeads。Array 用 minfi 或 ChAMP。

methylKit, DSS (Bayesian smoothing), bsseq, RnBeads. Arrays: minfi or ChAMP.

細胞 deconvolution

Bulk 甲基化常用 EpiDISH、methylCC 估算組織內細胞比例（血液最成熟，含 Houseman 參考）。

For bulk methylation, EpiDISH, methylCC deconvolve cell-type proportions (blood is most mature, e.g., Houseman reference).

表觀年齡

Horvath (353 CpG)、Hannum、PhenoAge、GrimAge、DunedinPACE (2022)、CausAge — 加速老化指標廣泛用於老化與疾病研究。

Horvath (353 CpG), Hannum, PhenoAge, GrimAge, DunedinPACE (2022), CausAge — accelerated-aging metrics widely used in aging / disease studies.

互動

三、DMR 偵測模擬

調整 Δβ 與 q-value 閾值，看「真實 DMR vs 雜訊」如何被分類。生物上常用 |Δβ| ≥ 10% (0.1) + q < 0.05。

Move Δβ and q-value cutoffs to see how "true DMRs vs noise" are classified. Biologically common: |Δβ| ≥ 10% (0.1) + q < 0.05.

四、實作

# 1. 建索引（一次性）
bismark_genome_preparation --bowtie2 GRCh38/

# 2. 比對
bismark --genome GRCh38/ -1 R1.fq.gz -2 R2.fq.gz \
        -p 8 --output_dir bm/

# 3. 去重複 + 甲基化擷取
deduplicate_bismark --paired bm/sample_pe.bam
bismark_methylation_extractor --paired-end --comprehensive \
        --bedGraph --genome_folder GRCh38/ bm/sample_pe.deduplicated.bam

# 4. 報告
bismark2report; multiqc bm/
# methylKit — DMR 偵測
library(methylKit)
files <- list("ctrl1.CpG", "ctrl2.CpG", "trt1.CpG", "trt2.CpG")
meth  <- methRead(files, sample.id=list("c1","c2","t1","t2"),
                  assembly="hg38", treatment=c(0,0,1,1),
                  context="CpG", mincov=10)
united <- unite(meth)

# DMP（單一 CpG）
dmp <- calculateDiffMeth(united, mc.cores=8)
sig <- getMethylDiff(dmp, difference=10, qvalue=0.05)

# DMR（合併鄰近 sig CpG）
tiles <- tileMethylCounts(meth, win.size=1000, step.size=1000)
dmr <- calculateDiffMeth(unite(tiles)) |>
       getMethylDiff(difference=10, qvalue=0.05)
# minfi — EPIC v2 array
library(minfi); library(IlluminaHumanMethylationEPICv2anno.20a1.hg38)
rgset <- read.metharray.exp(targets = pheno, force=TRUE)
gmset <- preprocessFunnorm(rgset)        # normalization
gmset <- gmset[!is.na(rowSums(getBeta(gmset))), ]

# DMP（site-level）
beta <- getBeta(gmset)
dmp  <- dmpFinder(beta, pheno = pheno$group, type="categorical")
sig  <- dmp[dmp$qval < 0.05, ]

# DMR（bump-hunting）
library(bumphunter)
dmr <- bumphunter(gmset, design=model.matrix(~pheno$group),
                  cutoff=0.10, B=1000)
