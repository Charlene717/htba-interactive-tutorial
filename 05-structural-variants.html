結構變異與長讀定序

短讀偵測不到的世界：SV、CNV、重複序列、T2T 與 pangenome graph。

The world short reads miss: SVs, CNVs, repeats, T2T genomes and pangenome graphs.

SV 類別

一、SV 六大類

DEL（缺失）

≥ 50 bp 被刪除。微缺失（< 50 bp）歸 indel。

≥ 50 bp deletion. Micro-deletions (< 50 bp) are indels.

DUP（重複）

區段被複製多份；tandem 或 dispersed。CNV 的常見來源。

Region duplicated; tandem or dispersed. A common source of CNVs.

INV（倒位）

區段翻轉方向。短讀偵測非常困難 — 雙端 read 仍 mapping 到同位點。

Region flipped. Very hard for short reads — both ends still map to the same locus.

INS（插入）

新片段插入（含 mobile elements LINE/SINE/SVA）。長讀 + de novo 組裝才能完整解析。

Novel insertion (including mobile elements LINE/SINE/SVA). Full resolution needs long reads + de novo assembly.

TRA / BND（易位）

染色體間或內部斷裂與重接（breakend）。腫瘤 fusion gene、淋巴瘤 BCR-ABL 經典案例。

Inter- or intra-chromosomal break-and-join (breakends). Classic in tumor fusion genes (e.g., BCR-ABL).

CNV（拷貝數變異）

區段拷貝數 ≠ 2（正常 diploid）。CNVkit / GATK CNV / Control-FREEC 透過 depth ratio + B-allele frequency 估計。

Copy number ≠ 2 (normal diploid). CNVkit / GATK CNV / Control-FREEC use depth ratio + BAF.

T2T 與 Pangenome

三、T2T 與 Pangenome

T2T-CHM13（2022 Science）：第一個完整、無 gap 的人類參考基因組（3.055 Gb），新增 200 Mb 序列含 1956 個新基因，徹底解開 centromere、acrocentric 短臂、rDNA 區。

Human Pangenome Reference Consortium (HPRC) 2023 Nature：用 47 個多元族群個體做的 graph-based 參考，比 GRCh38 多偵測 104% 的 SV、平均減少 34% 的 small-variant 錯誤。工具：vg、minigraph-cactus、PanGenie、vg giraffe。

2024-2025 進展：HPRC Year 2 完成 ~100 個人；MGI/CN 人類 pangenome (CPC) 提供更多東亞代表；T2T-Y (Nurk et al.) 補完 Y 染色體。

T2T-CHM13 (Science 2022): the first complete, gapless human reference (3.055 Gb), adding 200 Mb of sequence and 1956 new genes — fully resolving centromeres, acrocentric short arms, and rDNAs.

HPRC Pangenome (Nature 2023): a graph-based reference from 47 diverse individuals; detects 104% more SVs than GRCh38 and reduces small-variant errors by ~34% on average. Tools: vg, minigraph-cactus, PanGenie, vg giraffe.

2024-2025 progress: HPRC Year-2 reached ~100 genomes; the MGI/CN human pangenome (CPC) adds East-Asian diversity; T2T-Y (Nurk et al.) completed the Y chromosome.

五、SV 呼叫與合併

# Manta — Illumina short-read SV
configManta.py --bam sample.bam --referenceFasta ref.fa \
                --runDir mantaRun
mantaRun/runWorkflow.py     # 輸出 candidateSV.vcf.gz

# Sniffles2 — long-read SV
sniffles --input sample.hifi.bam --reference ref.fa \
         --vcf sample.sniffles.vcf --threads 16 \
         --tandem-repeats hg38.trf.bed   # TR-aware

# CNVkit — CNV from WES/WGS
cnvkit.py batch tumor.bam --normal normal.bam \
          --targets exons.bed --fasta ref.fa --output-dir cnv_out
# SURVIVOR — 多 caller ensemble
cat > vcf_list.txt <<EOF
manta.vcf
delly.vcf
sniffles.vcf
EOF

SURVIVOR merge vcf_list.txt 500 2 1 1 0 30 merged.vcf
# 參數：距離 500bp | 至少 2 caller 支持 | 同 type | 同 strand | dup-or-ins | ≥30 bp

# 與 dbVar / DGV 比對篩選 novel SV
bedtools intersect -v -a merged.vcf -b common_SV_dbvar.bed > novel.vcf
# R — 註釋與視覺化
library(StructuralVariantAnnotation)
library(VariantAnnotation)

vcf <- readVcf("merged.vcf", "hg38")
gr  <- breakpointRanges(vcf)

# 與 RefSeq 基因註釋交集 — 偵測 fusion 候選
library(TxDb.Hsapiens.UCSC.hg38.knownGene)
txdb <- TxDb.Hsapiens.UCSC.hg38.knownGene
hits <- findOverlaps(gr, genes(txdb))

# 用 gTrack / Sushi 畫局部結構
library(gTrack)
plot(gTrack("hg38", gr[1:10]))
