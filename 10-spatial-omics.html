空間轉錄組

Visium HD、Xenium、Stereo-seq、Slide-seq — 2025 平台對照 + 空間統計與 deconvolution。

Visium HD, Xenium, Stereo-seq, Slide-seq — 2025 platform overview + spatial statistics + deconvolution.

兩大派別

一、兩大派別

📚 Sequencing-based

把空間 barcode 印在玻片上，捕獲組織的 mRNA、做 NGS。全轉錄組、解析度由 spot/bin 大小決定。代表：Visium / Visium HD、Stereo-seq、Slide-seq v2、DBiT-seq。

Spatial barcodes printed on a slide capture tissue mRNA, then NGS. Whole-transcriptome, resolution set by spot/bin size. Representatives: Visium / Visium HD, Stereo-seq, Slide-seq v2, DBiT-seq.

🔍 Imaging-based

在原位以螢光探針偵測目標基因（padlock probe / branched DNA）。單細胞 / 亞細胞解析度、但只能偵測預先設計的 panel（百到千個基因）。代表：Xenium / Xenium Prime 5K、MERSCOPE、CosMx、seqFISH+。

In-situ detection with fluorescent probes (padlock or branched DNA). Single-cell / subcellular resolution but limited to pre-designed panels (hundreds to thousands of genes). Representatives: Xenium / Xenium Prime 5K, MERSCOPE, CosMx, seqFISH+.

💡
選型法則（2025 benchmark）：想要「整片組織 + 全轉錄組」用 Visium HD / Stereo-seq；想要「亞細胞 + 已知 panel」用 Xenium。Xenium 每基因敏感度約為 Visium HD 的 3.18×；Visium HD 偵測到的基因總數更多。常見策略：在同一張連續切片上做 Visium HD（廣度）+ Xenium（深度）兩者整合（見 Ch 16）。 Selection rule (2025 benchmarks): Whole tissue + whole-transcriptome → Visium HD / Stereo-seq; subcellular + known panel → Xenium. Per-gene sensitivity of Xenium ≈ 3.18× Visium HD; total gene detection is higher with Visium HD. Common strategy: Visium HD (breadth) + Xenium (depth) on serial sections, integrated via methods in Ch 16.
分析流程

三、分析流程

Upstream

Visium → Space Ranger；Xenium → Xenium Onboard Analysis；Stereo-seq → SAW。輸出：spot/cell × gene matrix + 空間座標 + H&E 影像。

Visium → Space Ranger; Xenium → Xenium Onboard Analysis; Stereo-seq → SAW. Output: spot/cell × gene matrix + spatial coords + H&E image.

QC + Normalization

SCTransform 或 sctransform-spatial。注意：含 anatomical 結構的組織會有強空間 autocorrelation，是訊號不是 batch 效應。

SCTransform or sctransform-spatial. Note: tissues with anatomical structure show strong spatial autocorrelation — that's signal, not batch.

空間 domain

BayesSpace、SpaGCN、STAGATE、GraphST（2023）— 結合 expression + 空間鄰居找出組織 domain。

BayesSpace, SpaGCN, STAGATE, GraphST (2023) — combine expression + spatial neighbors to find tissue domains.

SVG

SpatialDE、SPARK / SPARK-X、nnSVG（2023）— 找有空間 pattern 的基因，類似 HVG 但加上空間統計。

SpatialDE, SPARK / SPARK-X, nnSVG (2023) — find genes with spatial pattern; like HVGs but spatial-aware.

Deconvolution

Visium 一個 spot 涵蓋多個細胞，需用 cell2location、RCTD、SPOTlight、Tangram 推估細胞型態比例。詳見 Ch 16。

Visium spots cover multiple cells — use cell2location, RCTD, SPOTlight, Tangram to estimate cell-type fractions. See Ch 16.

五、實作

# Squidpy（與 Scanpy 整合）
import scanpy as sc, squidpy as sq
adata = sq.datasets.visium("V1_Mouse_Brain")

# QC + 標準預處理
sc.pp.filter_cells(adata, min_counts=500)
sc.pp.normalize_total(adata); sc.pp.log1p(adata)
sc.pp.highly_variable_genes(adata, n_top_genes=2000)
sc.pp.scale(adata); sc.tl.pca(adata)
sc.pp.neighbors(adata); sc.tl.leiden(adata, resolution=0.6)

# 空間關係 + SVG
sq.gr.spatial_neighbors(adata, coord_type="generic")
sq.gr.nhood_enrichment(adata, cluster_key="leiden")
sq.gr.spatial_autocorr(adata, mode="moran")    # Moran's I
sq.pl.spatial_scatter(adata, color=["leiden", "total_counts"])
# Seurat spatial workflow（Visium）
library(Seurat)
brain <- Load10X_Spatial("V1_Adult_Mouse_Brain/")
brain <- SCTransform(brain, assay="Spatial")
brain <- RunPCA(brain) |> FindNeighbors(dims=1:30) |>
         FindClusters(resolution=0.5) |> RunUMAP(dims=1:30)

SpatialDimPlot(brain, label=TRUE, label.size=3)
SpatialFeaturePlot(brain, features=c("Snap25","Hpca"))

# SpatiallyVariableFeatures（基於 Moransi）
brain <- FindSpatiallyVariableFeatures(brain, assay="SCT", selection.method="moransi")
# Giotto Suite — 跨平台空間分析
library(Giotto)
gobj <- createGiottoObject(expression="expr.mtx",
                          spatial_locs="coords.tsv")
gobj <- normalizeGiotto(gobj) |> calculateHVF() |>
        runPCA() |> runUMAP() |>
        createNearestNetwork(dimensions_to_use=1:15) |>
        doLeidenCluster(resolution=0.4)

# SVG via binSpect
svg <- binSpect(gobj, bin_method="kmeans")
# cell-cell interaction
cellProximityEnrichment(gobj, cluster_column="leiden")
