五大組學

一、五大組學版圖

當代生醫研究通常將分子層級切成五大「組學」。每一層皆有對應的高通量量測技術，後續章節會逐一深入；現在先建立全景視野。

Modern biomedical research stratifies molecular biology into five "omics" layers. Each has its own high-throughput readout — later chapters dive in; here we build the panoramic view.

🧬

Genomics

DNA 序列本身。WGS / WES / 靶向定序 / 長讀組裝。產出：FASTQ → BAM → VCF（SNV / indel / SV / CNV）。

The DNA sequence itself. WGS / WES / targeted / long-read assembly. Output: FASTQ → BAM → VCF (SNV / indel / SV / CNV).

🪟

Epigenomics

染色質可及性、TF binding、組蛋白修飾、DNA 甲基化。ATAC-seq、ChIP-seq、CUT&RUN、WGBS、EPIC array。產出：peaks、methylation β-values。

Chromatin accessibility, TF binding, histone marks, DNA methylation. ATAC-seq, ChIP-seq, CUT&RUN, WGBS, EPIC array. Output: peaks, methylation β-values.

📊

Transcriptomics

mRNA 量。Bulk / single-cell / spatial / long-read。產出：counts、UMI、splice junction、isoform。

mRNA abundance. Bulk / single-cell / spatial / long-read. Output: counts, UMI, splice junctions, isoforms.

🧪

Proteomics

蛋白質豐度與 PTM。LC-MS DDA / DIA、CITE-seq、Olink、SomaScan。產出：peptide → protein intensity、site-level PTM。

Protein abundance and PTMs. LC-MS DDA / DIA, CITE-seq, Olink, SomaScan. Output: peptide → protein intensities, site-level PTMs.

⚗️

Metabolomics

小分子代謝物。LC-MS / GC-MS / NMR、targeted vs. untargeted。產出：m/z–RT feature table，依賴譜庫鑑定。

Small-molecule metabolites. LC-MS / GC-MS / NMR, targeted vs. untargeted. Output: m/z–RT feature tables, library-dependent identification.

🦠

（延伸）Microbiome

16S rRNA、shotgun metagenomics、metatranscriptomics。產出：ASV / OTU table、MAG。常與其他 omics 共同整合。

16S rRNA, shotgun metagenomics, metatranscriptomics. Output: ASV/OTU tables, MAGs. Often integrated with other omics.

💡
核心原則：每個組學層只回答「分子層級的一面」，整合多層才能逼近完整的生物機制。Ch 14–18 將處理整合方法。 Core principle: Each omics layer answers only one molecular facet — true mechanism emerges only with integration. Ch 14–18 covers integration methods.
定序原理

二、短讀 vs. 長讀：核心 trade-off

特性 Illumina (SBS) PacBio HiFi (CCS) Oxford Nanopore (ONT)
讀長50–300 bp10–25 kb10–100 kb（最長 >4 Mb）
準確度>99.9%>99.9% (HiFi)~99%
錯誤模式substitution 為主random，已被 HiFi 大幅壓低indel 為主（homopolymer） Substitution-biasedRandom, largely reduced by HiFiIndel-biased (homopolymers)
通量NovaSeq X 25 Tb/runRevio 360 Gb/runPromethION 14 Tb/run
強項高精度、低 cost/Gb、SNV、表達量化de novo 組裝、SV、phasing、全長 isoform即時 (real-time)、超長片段、direct RNA、modification Accuracy, low $/Gb, SNVs, expression quantificationDe novo assembly, SVs, phasing, full-length isoformsReal-time, ultra-long reads, direct RNA, base modifications
⚠️
選型法則：表達量化、SNV 與目標重定序選 Illumina；新基因組組裝、結構變異與全長 isoform 選 HiFi；超長片段、即時診斷與 m6A/CpG 同步偵測選 ONT。 Selection rule: Use Illumina for quantification / SNV / re-sequencing; HiFi for de novo assembly, SV, and full-length isoforms; ONT for ultra-long reads, real-time diagnostics, and joint m6A/CpG detection.
通用資料流

三、所有組學共通的資料流

原始訊號 / Raw signal

儀器層級的影像 / 電流：Illumina .bcl、PacBio .subreads.bam、ONT POD5、MS .raw / .mzML。多半透過原廠軟體交付。

Instrument-level images / current: Illumina .bcl, PacBio .subreads.bam, ONT POD5, MS .raw / .mzML. Usually handled by vendor software.

FASTQ / spectra

序列文字 + 品質分數（Q-score）。MS 領域對應 mzML / mzXML。第一個 QC 點：FastQC、MultiQC、msstats QC。

Sequence text + quality (Q-score). MS analog: mzML / mzXML. First QC point: FastQC, MultiQC, msstats QC.

比對 / 鑑定

DNA/RNA → BAM/CRAM (bwa-mem2, STAR, minimap2)；MS → identified PSM (MSFragger, Comet)。

DNA/RNA → BAM/CRAM (bwa-mem2, STAR, minimap2); MS → identified PSMs (MSFragger, Comet).

量化結果 / Quantification

VCF (variants)、count matrix (RNA)、peak BED (ATAC/ChIP)、β-value matrix (Methyl)、intensity matrix (Proteomics/Metabolomics)。這是統計分析的真正起點。

VCF (variants), count matrix (RNA), peak BED (ATAC/ChIP), β-value matrix (Methyl), intensity matrix (Proteomics/Metabolomics). This is where statistical analysis actually begins.

生物學詮釋

DE 基因 / DMR / DAR、enrichment、pathway crosstalk、cell-cell communication、knowledge graph。Ch 17–18 詳述。

DE genes / DMRs / DARs, enrichment, pathway crosstalk, cell–cell communication, knowledge graphs. Detailed in Ch 17–18.

公開資料庫

五、必須認識的公開資料庫

🌐 通用定序資料

SRA (NCBI)、ENA (EBI)、DDBJ：原始 FASTQ；三者每日鏡像同步。
GEO / ArrayExpress：含 metadata 的 series-level 表達資料。
下載：sra-toolkit、ffq、nf-core/fetchngs。

SRA (NCBI), ENA (EBI), DDBJ: raw FASTQ; the three sync daily.
GEO / ArrayExpress: series-level expression with metadata.
Download via sra-toolkit, ffq, nf-core/fetchngs.

🏥 臨床 / 大型 cohort

TCGA / PCAWG / GENIE（癌症）、GTEx（正常組織 RNA-seq）、UK Biobank / All of Us（族群基因組）、dbGaP / EGA（受控存取）。商業使用需查 DUA。

TCGA / PCAWG / GENIE (cancer), GTEx (normal-tissue RNA-seq), UK Biobank / All of Us (population genomics), dbGaP / EGA (controlled access). Check DUA for commercial use.

🔬 專題資料庫

ENCODE / Roadmap Epigenomics：表觀。
HCA / CELLxGENE / Tabula Sapiens：scRNA。
PRIDE / MassIVE：蛋白質。
Metabolights / MetabolomicsWorkbench：代謝體。
AlphaFold DB：預測結構。

ENCODE / Roadmap Epigenomics: epigenetics.
HCA / CELLxGENE / Tabula Sapiens: scRNA.
PRIDE / MassIVE: proteomics.
Metabolights / MetabolomicsWorkbench: metabolomics.
AlphaFold DB: predicted structures.

📑 知識與參考

Ensembl / GENCODE：基因模型。
UniProt：蛋白質。
gnomAD / ClinVar / OMIM：變異族群頻率與臨床顯著性。
Reactome / KEGG / WikiPathways / MSigDB：通路與 gene sets。

Ensembl / GENCODE: gene models.
UniProt: proteins.
gnomAD / ClinVar / OMIM: population frequencies and clinical significance.
Reactome / KEGG / WikiPathways / MSigDB: pathways and gene sets.

FAIR

六、FAIR 原則

🌳 FAIR = Findable · Accessible · Interoperable · Reusable

F
Findable：給資料一個全球唯一的 DOI / accession（如 GSE12345），加上豐富的 metadata，並登錄索引（GEO、ENA、Zenodo）。
A
Accessible：用標準協定取得（HTTPS、SFTP、AWS S3），即使原始資料 retire 也保留 metadata。受控存取請使用 dbGaP / EGA。
I
Interoperable：用社群標準格式（FASTQ、CRAM、VCF、mzML、AnnData h5ad）與標準字彙（HPO、MeSH、UBERON、SNOMED-CT、EFO）。
R
Reusable：明確授權（CC BY、CC0、控管存取）、處理流程版本化（git + Snakemake/Nextflow）、容器化（Docker / Singularity）。
F
Findable: Give each dataset a globally unique DOI / accession (e.g., GSE12345), rich metadata, and register it in an index (GEO, ENA, Zenodo).
A
Accessible: Retrieve via standard protocols (HTTPS, SFTP, AWS S3); keep metadata even if raw data is retired. Use dbGaP / EGA for controlled access.
I
Interoperable: Use community-standard formats (FASTQ, CRAM, VCF, mzML, AnnData h5ad) and vocabularies (HPO, MeSH, UBERON, SNOMED-CT, EFO).
R
Reusable: Explicit license (CC BY, CC0, controlled), version-controlled pipelines (git + Snakemake/Nextflow), containerized (Docker / Singularity).

七、從公開資料庫下載資料

# R：下載 GEO series + soft 檔 / Download GEO series + SOFT
if (!requireNamespace("BiocManager", quietly=TRUE)) install.packages("BiocManager")
BiocManager::install(c("GEOquery", "SummarizedExperiment"))
library(GEOquery)

gse <- getGEO("GSE183947", GSEMatrix = TRUE)[[1]]
pData(gse) |> head()       # 樣本 metadata
exprs(gse) |> dim()        # 表達矩陣

# 直接抓 supplementary 原始 counts
getGEOSuppFiles("GSE183947", baseDir = "./data")
# Python：用 ffq 取得跨資料庫元資料 → 用 prefetch 下載
# pip install ffq
import json, subprocess

meta = subprocess.check_output(["ffq", "GSE183947"])
meta = json.loads(meta)
print(meta["GSE183947"]["samples"].keys())

# 取 SRR 列表後再用 sra-toolkit 下載
srrs = [s["runs"] for s in meta["GSE183947"]["samples"].values()]

# 進階：HCA / CELLxGENE 用 cellxgene-census 直接讀取
import cellxgene_census
with cellxgene_census.open_soma() as census:
    adata = cellxgene_census.get_anndata(
        census, organism="Homo sapiens",
        obs_value_filter="tissue == 'lung' and disease == 'normal'")
print(adata.shape)
# Shell：使用 sra-toolkit + pigz 平行解壓
# conda install -c bioconda sra-tools pigz
prefetch SRR15852396                            # 下載 .sra
fasterq-dump --split-files --threads 8 SRR15852396   # 轉 FASTQ
pigz -p 8 SRR15852396_*.fastq                  # 平行壓縮

# 更現代的選擇：nf-core/fetchngs（自動處理 metadata 與 md5）
